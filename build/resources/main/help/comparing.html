Comparing Diagrams

If you click on "Diagram/Comparing Diagrams", with a diagram selected, you will be prompted to select another diagram file to compare against, using the FileChooser dialog.

For certain types of mismatch, a separate frame will apear with more information - this will disappear, along with the mismatch indicators (see below), when you select "Diagram/Clear Visible Indicators".

When you have selected one, you will see the current diagram with a number of blocks and/or arrows marked with letter codes in circles.

The letter codes are as follows:

  • "A" - the block or arrow has been added
  • "O" - the arrow has been omitted
  • "C" - the block or arrow has been changed in some way
  • "M" - the block has been moved (this does not apply to arrows)
